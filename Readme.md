## Install

```
npm i updatelaodhost -g
```

## Method 1:

```
updatelaodhost --url=https://??? --password=???
```

## Method 2:

use default url(https://laod.cn)

```
updatelaodhost  --password=???
```

## Method 3:

use default url and Export(path)

```
export SUDO_PASSWORD='???'

updatelaodhost
```
